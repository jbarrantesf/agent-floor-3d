🎯 HOW TO RUN (QUICK START)
═════════════════════════════════════════════════════════════════════════════════

1. Clone repository:
   git clone https://github.com/jbarrantesf/agent-floor-3d.git
   cd agent-floor-3d

2. Install dependencies:
   npm install

3. Configure environment:
   cp .env.example .env
   # Add Supabase credentials

4. Run all tests:
   npm test

5. Start development server:
   npm run dev

6. Open browser:
   http://localhost:5173

7. You should see:
   - 3D floor with 3 agent avatars
   - Real-time task flow animations
   - Cost ticker updating
   - Control panel (right side)
   - Interactive controls enabled
