🏗️ ARCHITECTURE
═════════════════════════════════════════════════════════════════════════════════

AGENT FLOOR 3D: Multi-Agent Orchestration System
┌─────────────────────────────────────────────────────────────────┐
│ HERMES (Orchestrator)                                           │
│  • TaskManager.ts — Delegates tasks to ORBIT                    │
│  • Monitors agent capacity & load                               │
│  • Receives results in real-time                                │
│  • Tracks costs and performance                                 │
└────────────────────┬──────────────────────────────────────────┘
                     │ Task Delegation (via Supabase)
                     ↓
┌─────────────────────────────────────────────────────────────────┐
│ ORBIT (Executor / Router)                                       │
│  • TaskQueue.ts — Polls Supabase for tasks                      │
│  • SubagentRouter.ts — Intelligent task distribution            │
│  • 5 Executor classes — File, SQL, Shell, Webhook, HTTP        │
│  • Load balancing & capacity management                         │
│  • Reports results back to Hermes in real-time                  │
└────────────────────┬──────────────────────────────────────────┘
                     │ Task Distribution
                     ↓
      ┌──────────────┼──────────────┐
      ↓              ↓              ↓
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│ Subagent 1   │ │ Subagent 2   │ │ Subagent 3   │
│ (File, Shell)│ │ (SQL, HTTP)  │ │ (Shell, HTTP)│
│              │ │              │ │              │
│ Parallel     │ │ Parallel     │ │ Parallel     │
│ Execution    │ │ Execution    │ │ Execution    │
└──────────────┘ └──────────────┘ └──────────────┘

Communication:
  Hermes ←→ ORBIT ←→ Subagents ← All via Supabase + Real-time Subscriptions

🚀 HOW TO GET STARTED
═════════════════════════════════════════════════════════════════════════════════

1. Clone the repo:
   git clone https://github.com/jbarrantesf/agent-floor-3d.git

2. Install deps:
   npm install

3. Configure Supabase:
   cp .env.example .env
   # Add your Supabase credentials

4. Run tests:
   npm test

5. Start Hermes (orchestrator):
   npm run hermes

6. Start ORBIT (executor + router):
   npm run orbit

7. Run an end-to-end demo:
   npm run demo
